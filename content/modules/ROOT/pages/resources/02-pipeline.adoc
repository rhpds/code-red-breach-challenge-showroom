== Components

Since last year's security issue, the pipeline security was re-enforced, and new standards emerged:

1. All pipelines are automated and are triggered by webhooks.

2. Every source code repository is private and accessible only via SSH Keys.

3. All build images need to come from 'registry.access.redhat.com'


The pipelines were configured to support those standards.

image::ctf-pipeline-ssh.png[Pipeline structure,link=self,window=_blank]
